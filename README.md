# journal-rest-api

A simple journal app using Nodejs, Express, tailwindCSS.

### To Start The App

- In the 'backend' folder run `npm install` then `npm start`.

- Then, in the 'frontend' folder open an html file from 'dist' folder.

#### Client is able to:

- Display all posts
- Add new post
- Edit post
- Delete post
- Search for posts by title or content
